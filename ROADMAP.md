# Roadmap

The roadmap for **lukk-js** (this client) and **lukk** (the Laravel package) — planned work plus the
shipped feature inventory — lives on the docs site, so both are in one place:

📍 **[stsepelin.github.io/lukk-docs/roadmap](https://stsepelin.github.io/lukk-docs/roadmap)**

Nothing there is a commitment; everything ships opt-in and backward-compatible. Have a use case that
needs a planned item — or something not listed? [Open an issue](https://github.com/stsepelin/lukk-js/issues).
Shipped work is detailed in each package's `CHANGELOG.md`.
